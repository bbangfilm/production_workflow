03 현장 노출 세팅 워크플로우 — 순서대로 진행
01
조명 세팅 완료 확인
조명팀이 최종 조명 위치 및 밸런스를 완료한 후 진행합니다. 조명이 변경되면 이 과정을 처음부터 다시 반복합니다.
02
카메라 EI 설정
밝은 환경(충분한 조명): EI 400 (Base 800)
어두운 환경 / 분위기 씬: EI 1600 (Base 3200) — BURANO 기준
A-CAM과 B-CAM 모두 동일한 Base × EI 조합으로 통일합니다.
03
18% 그레이카드 배치
피사체 위치의 메인 라이트 방향으로 카드를 세웁니다. 카드가 뷰파인더에 꽉 차도록 근접 촬영하거나 조감독이 카드를 프레임에 고정합니다. 카드의 그림자가 생기지 않도록 주의합니다.
04
웨이브폼으로 노출 조정 (A-CAM 기준)
웨이브폼(퍼레이드 또는 루마 모드)에서 그레이카드의 파형 위치를 확인합니다.
조리개 또는 조명 광량으로 파형을 아래 목표 위치에 맞춥니다.
뷰파인더 이미지가 "밝아 보이는 것"이 정상입니다 — 수치로 판단합니다.

46~48 IRE  —  IRE / % 표시 모니터 (SmallHD 등 일반 설정)
470~495 CV  —  Code Value 표시 모니터 (10-bit / 0~1023 풀 레인지 기준)
322~337 mV  —  mV 표시 모니터 (0~700mV 스케일 기준, 드물게 사용)
05
하이라이트 클리핑 확인
팔스컬러 또는 웨이브폼 상단을 확인합니다.
하이라이트(흰색 배경, 하이라이트 조명 등)가 90 IRE 이하에 위치해야 합니다.
클리핑이 발생하면 조명 광량을 줄이거나 ND 필터를 추가합니다.
EI 400은 기본 헤드룸이 1스탑 줄어든 상태임을 항상 염두에 둡니다.
06
팔스컬러로 스킨톤 확인
실제 배우/모델의 피부가 프레임 안에 있을 때 팔스컬러를 켭니다.
EI 400 세팅에서 스킨톤은 팔스컬러 기준으로 Pink~Red 존에 위치해야 합니다 (포스트에서 정상으로 돌아올 값).
팔스컬러 툴에 따라 색 기준이 다를 수 있으므로 첫 번째 사용 전 캘리브레이션을 권장합니다.
07
B-CAM (FX3) 노출 매칭
FX3도 동일하게 EI 400으로 설정 후, 웨이브폼 기준 46~48 IRE에 맞춥니다.
두 카메라의 노출이 웨이브폼 기준으로 일치하는지 A/B 모니터를 나란히 두고 비교합니다.
렌즈 특성에 따라 T스톱 편차가 있을 수 있으니 조리개가 아닌 웨이브폼 수치를 매칭 기준으로 삼습니다.
08
화이트 밸런스 확인
그레이카드를 활용하여 A-CAM / B-CAM 수동 WB를 동일하게 맞춥니다.
컬러스코프(벡터스코프)에서 그레이카드가 중앙에 위치하는지 확인합니다.
두 카메라 WB 불일치는 포스트에서 매칭 작업이 추가되므로 현장에서 반드시 통일합니다.
09
최종 기록 후 촬영 시작
슬레이트(클랩보드)에 현재 사용 중인 Base ISO / EI 값을 기재합니다 (예: Base 800 / EI 400).
씬 시작 전 그레이카드 샷을 짧게 녹화해 두면 포스트에서 노출 기준 확인에 유용합니다.
